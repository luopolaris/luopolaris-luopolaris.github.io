二、什么是HTML?

HTML:超文本标记语言。（Hypertext、Markup、language）

超文本：是指页面可以包含图片、链接、音乐等非文字元素。

<a>超链接

属性：href=”需要打开的地址”（必须有的属性）

打开外网时，地址必须加上“http:// ”

<b>加粗

　　例：<b>加粗</b>不加粗

<i>斜体

　　例：<i>斜体</i>非斜体

<u>下划线

　　例：<u>下划线</u>不加下划线

<br> <p>两个标签都有换行的作用

<img>图片

元素语法：
1.以开始标签起始。

2.以结束标签终止。

3.元素内容是开始标签和结束标签里面的内容。

4.某些HTML元素具有空内容（empty content）。

5.空元素在开始标签中进行关闭（以开始标签的结束而结束）。
